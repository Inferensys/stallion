# Stallion Case Study Assets

This directory contains the public demo assets for Stallion.

Stallion is a reference implementation for multi-agent orchestration: plan review, agent task ownership, parallel execution, sandbox preview, generated workspace files, and enterprise controls.

## Scenario

Task:

> Build a product analytics dashboard for a B2B SaaS company. Include acquisition, activation, retention, and revenue metrics, charts, and implementation notes.

The flow shows:

- discovery and readiness scoring
- human review before execution
- four specialist agents
- task dependencies and parallel execution
- live runtime preview
- workspace file inspection
- admin, team, billing, integration, audit, and settings surfaces
